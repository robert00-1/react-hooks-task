# Title
Tak Manager
# Description
A simple task manager built with react, demonstrating global site management with `usecontext`.Users can add task, mark them complete,and serch/filter tasks.
# Tools
vite
react-hook
javascript

# Setup
follow this steps to access and run  task manager application

clone repository into a local machine
git clone https://github.com/robert00-1/react-hooks-task.git

navigate into the folder
cd react-hooks-task
install dependancies
npm install
npm run dev
copy and paste the link that will appear on your terminal and run it on your browser to acess the manager task
